# 手冲手记 · UI/UX 优化设计

日期：2026-06-05
方向：保持现有手帐/实验记录本风格（奶油纸+咖啡棕+铜壶橘），重点优化交互质感与布局呼吸感。

## 1. 交互体系（"温润质感"）

### 原则
- 所有动画 150–350ms，ease-out / spring 曲线
- prefers-reduced-motion 用户跳过非必要动画
- 每个动画传达状态变化或引导注意力，不纯装饰

### 1.1 弹窗动画
- 打开：scale(0.92→1.0) + opacity(0→1)，300ms cubic-bezier(0.34, 1.56, 0.64, 1)
- 关闭：scale(1.0→0.95) + opacity(1→0)，150ms ease-in
- 背景遮罩：opacity 渐入 200ms

### 1.2 卡片悬停
- 浮起动画用 spring 曲线，hover 时 translateY(-4px) + 阴影加深
- 左缘铜橘线保留；新增边缘从 hairline→accent 过渡
- 冲煮卡 hover 时粉水比数字 scale(1.05)

### 1.3 标签页切换
- 底部指示条平滑滑动（left transition 0.3s cubic-bezier）
- 内容区 fade-up 从 0.4s 调为 0.35s

### 1.4 评分交互
- 自定义 range 滑块轨道：左暖色渐变→右中性灰
- 拖拽时 output 数字弹跳（scale 1.15 + 闪 accent）
- 滑到 8+ 轨道末端出现金色星标
- 数值显示改用大数字+小单位

### 1.5 删除操作
- 去掉原生 confirm() → 内联确认条（从卡片原位展开）
- 确认后卡片淡出收缩再移除
- 删除成功 toast 滑入，3s 消失

### 1.6 表单交互
- 聚焦 ring 加过渡动画
- 必填未填时标签红星微抖
- 粉水比/养豆预览数字变化时脉冲动画

### 1.7 列表入场
- 卡片交错动画（每个延迟 40-60ms）
- 仅首次加载和标签切换触发
- IntersectionObserver 按需触发

### 1.8 微反馈
- 保存按钮：短暂变绿+勾号
- 同步状态：文字闪绿 2s
- 导出：toast 滑入
- 豆子余量进度环

## 2. 布局重排（"呼吸感"）

### 2.1 豆子卡片
- 卡片最小宽度从 260px → 280px，间距从 20px → 24px
- 卡片内部 padding 从 24px → 28px
- 养豆天数数字加大到 28px
- 新增"余量进度环"（已用/总量）

### 2.2 冲煮记录
- 卡片 padding 从 20px → 24px
- 卡片间距从 16px → 20px
- 粉水比数字区域从 100px → 110px
- 移动端时间线式布局（左侧竖线+圆点代替三栏）

### 2.3 详情页
- 深色头部 padding 从 28px → 36px
- 统计数字加大
- 最佳配方卡片和调试轨迹之间加更多留白
- 返回按钮加面包屑感

### 2.4 弹窗表单
- 弹窗宽度从 640px → 680px
- 表单分组加视觉分隔（细线+标签）
- 必填标记更明显
- 按钮区加顶部分隔线

### 2.5 页面整体
- content 区 max-width 从 1080px → 1120px
- 顶部 header padding 增加
- 使用 CSS gap 统一间距

## 3. 云同步 UI 优化

### 3.1 同步面板重构
- 从独立行 → 融入 header 右侧工具栏
- 用图标+状态指示器代替文字面板
- 状态展示：未启用（灰）→ 同步中（转圈）→ 已同步（绿勾）

### 3.2 同步码设置
- 首次设置：小弹窗输入同步码
- 已设置：点击状态图标 → 下拉菜单（查看同步码/重新同步/断开）

### 3.3 同步反馈
- 同步中：header 上的小转圈动画
- 成功：状态图标短暂变绿
- 失败/离线：状态图标变灰+提示文字
- 冲突：静默合并，冲突条目 toast 提示

## 4. 不做的（明确排除）
- 首页仪式感大改
- 内置冲煮计时器
- 风味轮
- 日/夜模式
- 色板大改
