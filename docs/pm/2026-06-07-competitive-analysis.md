# 竞品分析：手冲手记（咖啡冲煮记录 + 配方优化）

**日期**：2026-06-07
**分析对象**：5 个直接竞品 + 3 个间接竞品（国际 + 国内两个战场）
**镜头**：定位 + 功能差异化（回答"凭什么是你"）
**用途**：收敛价值主张、指导产品策略

> 名词速记：
> - **直接竞品** = 同品类、抢同一批用户（都做"咖啡记录"）。
> - **间接竞品** = 用别的方式解决同一个需求（比如"教你冲好咖啡"也能让人不需要记录）。
> - **Table stakes（入场门票）** = 不做就没资格上桌，但做了也不会赢。
> - **Differentiator（差异点）** = 真正让人选你、记住你的东西。

---

## 市场概览

咖啡冲煮记录是个**全球都有、但高度分散**的小众品类。它分成两类玩家：
1. **记录派**：帮你把每杯参数存下来（Beanconqueror、享咖小记、咖啡计时器）。
2. **指导派**：给你现成的"冠军配方/步骤教程"照着冲（Filtru、Coffee.guru、趣点咖啡）。

关键趋势：免费 + 开源（Beanconqueror）已经把"纯记录"做到了天花板，且无广告无付费墙——**靠"记得更全"很难赢**。同时，几乎**没有人**真正占住第三块：**"把你自己的记录数据反过来变成你自己的最优配方建议"**。这正是空位。

### 间接竞品（同样解决"想喝到稳定好喝的手冲"）
- **趣点咖啡 / 咖啡工具箱**：知识教程 + 工具库，插画科普，不做个人记录。
- **微信备忘录 / Excel / 纸笔**：最大的隐形竞品——大多数人其实在用"随手记"。你要打败的首先是"懒得记"。

**核心洞察**：左上角"**低门槛 + 主动给你专属建议**"这个象限，目前基本是空的。手冲手记天然站在这里——这就是"凭什么是你"的答案。

## 差异化机会（按可防御性排序）

1. **🎯 你的私人配方教练（最强护城河）**
   别人是"日记本"，你是"顾问"。同样的记录数据，你多做了一层**分析 + 建议**：自动找高分配方共性、味道诊断给调整方向、最佳配方一键复现。
   *为什么可防御*：这是算法/产品逻辑层，不是堆字段。竞品要补得重写分析引擎，且与它们"纯记录"的定位冲突。

2. **⚡ 零摩擦入口（增长护城河）**
   打开网址即用，不下载、不注册。对"想试试但不想为此装个 App"的好奇用户，转化门槛几乎为零；在国内应用商店分裂、iOS 限制多的环境下尤其值钱。
   *为什么可防御*：竞品的原生 App + 账号体系是"沉没成本"，它们很难自废武功改成网页优先。

3. **🔁 闭环味道诊断（体验护城河）**
   "勾'发酸'→立刻告诉你下次怎么调"，把"记录"这个被动动作变成"我马上能改进"的主动动作。这是把记录变成习惯的钩子。

## 竞争威胁

1. **Beanconqueror 补上"分析建议"**：它开源、活跃、社区强。一旦它做"自动找规律"，会直接打到你的核心。
   *应对*：趁它还是"纯记录"，快速把"教练"心智占住，并把体验做得比硬核工具友好得多（它的复杂度是你的机会）。

2. **国内原生 App 有应用商店分发 + 可能的社交**：享咖小记等能被搜到、能晒圈。你是网页，**没有应用商店流量入口**。
   *应对*：把分享卡片（你已有 share.js）做成增长回路；靠内容/口碑而非商店分发。

3. **最大对手其实是"懒得记"**：用户用备忘录或干脆不记。
   *应对*：把首次记录做到 30 秒内完成，并让第 3 杯就出现"啊哈"洞察（见后续留存分析）。

## 建议

- **加倍投入（你赢的地方）**：
  "你的私人配方教练"——分析引擎 + 闭环诊断 + 最佳配方复现。所有对外表达都应围绕这一句，而不是"又一个记录本"。

- **补齐门票（避免被一票否决）**：
  ① 多设备同步的体验门槛（同步码对新手偏生涩，数据易丢是口碑杀手）；
  ② 给纯新手一点"起步配方/默认值"（借鉴 Filtru，但只作为冷启动脚手架，不喧宾夺主）。

- **暂时忽略（别被带跑）**：
  蓝牙电子秤集成、意式 espresso、教程内容库、社区社交。这些是竞品的主场，现在跟进会稀释你的"教练 + 零门槛"定位。
