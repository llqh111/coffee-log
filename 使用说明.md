# ☕ 手冲手记 · 使用说明

一个记录手冲咖啡变量的小软件。纯本地、不联网、不上传，数据只存在你这台电脑里。

## 怎么打开

直接**双击 `index.html`**，用浏览器（Chrome / Edge 都行）打开即用。
不需要安装任何东西，也不需要联网。

> 想放到桌面方便点：右键 `index.html` → 发送到 → 桌面快捷方式。

## 怎么用

### 1. 先加一包豆（「咖啡豆」标签）
点 **+ 新增豆子**，填名称、烘焙日期、产地等。
- 填了**烘焙日期**，软件会自动算「**养豆天数**」（今天 − 烘焙日），每天自动更新。

### 2. 记录每一杯（「冲煮记录」标签）
点 **+ 记录一杯**：
- 先选这杯用的是哪包豆；
- 填粉量、水量 → **粉水比自动显示**（如 1:15.0）；
- 还能记水温、研磨度、闷蒸、总时间、器具；
- 给这杯**打分（1–10）**并写**风味笔记**（强烈建议写，以后靠它找规律）。

### 3. 对比同一包豆的不同冲法
在「咖啡豆」标签**点任意一包豆**，进入详情页：
- 看到这包豆的平均分、最高分；
- 下面按**评分高→低**列出它的所有冲煮，方便看出「哪次参数更好喝」。

### 4. 看「分析」找规律
切到「**分析**」标签（要先给几杯打过分）：
- **🏆 高分配方**：自动挑出你评分最高的那几杯，总结它们的水温、粉水比、研磨度等集中在哪；
- **💡 下次试试**：根据高分杯，给一句具体的调参建议；
- **⚖️ 分类对比**：按器具、处理法分组比平均分，看哪类你打分更高。
- 顶部下拉可切「全部豆子」或单看某一包。打分的杯子太少时，会提示「仅供参考」——多记几杯结论才准。

## 备份与换电脑

- 点右上角 **导出备份** → 得到一个 `coffee-log-日期.json` 文件，妥善保存。
- 换电脑或清了浏览器缓存后，点 **导入** 选这个文件即可恢复。

> ⚠️ 数据存在浏览器本地。**换浏览器、清缓存、用"无痕模式"都会读不到原数据**，所以请养成偶尔导出备份的习惯。

## 文件说明（给好奇的你）

| 文件 | 作用 |
|---|---|
| `index.html` | 页面骨架（看得到的结构） |
| `style.css` | 样式（颜色、排版、好看的部分） |
| `app.js` | 逻辑（记录、计算、存取数据的大脑） |

三个文件要放在**同一个文件夹**里，缺一不可。想改颜色，打开 `style.css` 最上面的 `:root` 区域改几个色值即可。
