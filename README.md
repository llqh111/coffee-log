# ☕ 手冲手记 · 咖啡冲煮记录本

记录每一杯手冲咖啡的变量——咖啡豆、养豆天数、研磨度、水温、粉水比、评分与风味笔记，并自动**找出你的高分配方规律**。

纯前端网页应用：零安装、零联网即可用，数据只存在你自己浏览器本地（支持导出/导入备份）。已做成 **PWA**，手机可「添加到主屏幕」像 App 一样使用、并可离线。

## 🌐 在线使用

👉 **https://llqh111.github.io/coffee-log/**

手机浏览器打开上面的网址即可使用。想像 App 一样用：
- **安卓 Chrome**：菜单 → 「添加到主屏幕 / 安装应用」
- **iPhone Safari**：分享按钮 → 「添加到主屏幕」

## 功能

- **咖啡豆**：名称、烘焙商、烘焙日期（自动算养豆天数）、产地/处理法/烘焙度、克数/价格
- **冲煮记录**：粉量/水量（自动算粉水比）、研磨度、水温、闷蒸、总时间、器具、1–10 评分、风味笔记
- **单豆详情**：该豆所有冲煮按评分排序对比，平均分/最高分一目了然
- **评分分析**：高分配方共性、「下次试试」建议、按器具/处理法分组比平均分（数据少时如实提示「仅供参考」）

## 隐私

所有记录保存在你浏览器的 localStorage，**不上传任何服务器**。每个人用的是自己的一份，互不可见。换设备就用「导出备份 / 导入」搬数据。

## 技术

原生 HTML + CSS + JavaScript，无框架、无构建步骤。`index.html` / `style.css` / `app.js` + PWA 三件套（`manifest.webmanifest` / `sw.js` / `icon.svg`）。
