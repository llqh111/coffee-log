# ☕ coffee-log

一杯好咖啡，值得被记住。

咖啡记录工具已经够多了，但大多数要么太复杂，要么太简陋。**coffee-log** 踩在中间——它不假装你是 Q-Grader，也不假设你只喝速溶。你记录每一次冲煮，它帮你找出规律：为什么上次那杯那么好喝？这次的涩感是哪来的？

**[🌐 立即使用](https://coffee-log-b2o.pages.dev/)** · 全部数据存本地，不用注册。

## 它做了什么

### 豆子管理
记录每支豆子的产地、处理法、烘焙日期。烘焙日期不是摆设——它会帮你算出养豆期，提醒你这支豆子什么时候进入最佳风味期。

### 冲煮记录
每次冲煮记下参数：粉量、水量、水温、研磨度、冲煮时间。粉水比自动算，不用你掏出计算器。

### 风味诊断（核心功能）
喝完打分后，勾选你喝到的瑕疵味——**偏酸、偏苦、太淡、太重、涩感**。App 不会丢给你一篇"手冲入门指南"，它会针对你勾的问题，给具体的调整建议。比如"研磨度调粗半格，水温降到 88°C 试试"。

这是 coffee-log 和其他工具最大的区别：**不是记录你喝了什么，是帮你喝得更好。**

### 评分分析
给每杯咖啡打个分（1-10），它会自动找出你打分最高的那批冲煮，分析它们的共同点。比如"你给分最高的 5 杯，全是 92°C 水温、1:16 粉水比"——这就是你的最佳配方。

### 调试时间线
按时间排列每次冲煮的参数变化轨迹。什么时候调了研磨度，什么时候换了水温，一目了然。配合评分波动看，就是一张你自己动手优化咖啡的路径图。

### 冲煮计时器
内置分段计时器，闷蒸、注水、等待、完成，每阶段独立计时。不用再开手机秒表。

### 分享卡片
冲得满意的那杯，生成一张分享卡片发到群里。数据好看，排版也不丢人。

### 云端同步
扫码就能在不同设备间同步数据。用了墓碑机制（tombstone），不会出现删除的豆子又复活这种破事。同步是**可选的**，你完全可以只当本地工具用。

## 技术上的事

纯原生写的——HTML、CSS、JavaScript，没有框架，没有构建工具。

后端是 Cloudflare Pages Functions + KV，前端是 PWA，可以装到手机主屏离线用。数据默认存 localStorage，你什么时候想同步了再开。

为什么不用框架？因为目标是**零依赖、加载快、好维护**。一个 HTML 文件丢到服务器上就能跑。Core Web Vitals 全绿。

## 快速上手

1. 打开 [coffee-log-b2o.pages.dev](https://coffee-log-b2o.pages.dev)
2. 点「添加豆子」录入你手头的豆子
3. 冲一杯，点「记录冲煮」填参数
4. 喝完打分，勾选喝到的风味问题
5. 看看调整建议，下次试试
6. 重复 5 次以上，去「分析」页看你的最佳配方

没豆子？点「加载演示数据」先体验一遍流程。

---

## 隐私

所有数据默认存在你的浏览器里。**没有服务器能看到你的冲煮记录。** 云端同步是可选的，启用时数据用同步码加密传输。匿名统计数据只记录功能使用情况（访问、首次记录、完成5杯、开启同步），不碰任何冲煮参数或个人身份信息。
